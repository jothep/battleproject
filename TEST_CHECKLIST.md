## 🐛 常见问题排查

### 问题 1：ResourceManager 未找到
**错误信息：** `Invalid get index 'get_skill' on base: 'Nil'`

**解决方案：**
- 确认 `project.godot` 中 ResourceManager 已注册
- 重启 Godot 编辑器

### 问题 2：技能数据加载失败
**错误信息：** `技能不存在: skill_1`

**解决方案：**
- 检查 `data/skills/` 目录下的 JSON 文件
- 确认 JSON 格式正确（使用在线 JSON 验证器）

### 问题 3：角色创建失败
**错误信息：** `无法加载角色数据: p1`

**解决方案：**
- 检查 `data/characters/` 目录下的 JSON 文件
- 确认 `character_id` 字段正确

### 问题 4：DebugConfig 未定义
**错误信息：** `Identifier 'DebugConfig' not declared`

**解决方案：**
- 确认 `scripts/core/DebugConfig.gd` 文件存在
- 确认在 `project.godot` 中已注册为 Autoload
- 重启 Godot 编辑器
